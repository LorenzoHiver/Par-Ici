* Date de départ: date 
* Date de retour : date 
* Nombre de jours pendant le séjour : integer 
* Prénom : text

* Tel : char(10)
* Code : char(6)


### TABLE AVIS: 
* Date de création
* Prénom : text 
* Nom : text 
* Image : text(format à confirmer)
* Commentaire : varchar(300)
* Note : integer
